# Walkthrough - Release v1.2.0

The `APIRunner` project has been updated and released as version **1.2.0**. This release focuses on making the CSV parsing engine more robust.

## Changes Made

### 1. Robust CSV Parsing
- Replaced basic `String.Split(',')` with the powerful `CsvHelper` library.
- Safely handles tricky cases like internal commas and quoted strings without breaking (e.g., `"Smith, John"`).
- Automatically maps CSV rows directly into a strongly-typed `CsvRecord` C# object.
